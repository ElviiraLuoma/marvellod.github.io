RDF Representation using Turtle

We decided to use Turtle as a serialization of RDF, and chose to represent our two creators, one of their books, and one of the objects inpired by that book using RDF. You can see the completed and connected file below. To view a graph representation of this RDF, click here.

As can be seen in the RDF file, the URIs created for our people/objects are as follows. They each redirect to the data representation page dedicated to them.

  • https://marvellod.github.io/person/roald_dahl
  • https://marvellod.github.io/person/quentin_blake
  • https://marvellod.github.io/object/matilda
  • https://marvellod.github.io/object/sculpture

In addition to linking the same authorities for names (people, places, concepts) through owl:sameAs, we have connected our data to other related items that share different features and we have defined the following semantic associations. We widely used owl:sameAs and foaf:isPrimaryTopicOf but also used different properties and standards such as core:notablyAssociatedWith and foaf:primaryTopic. By doing this were able to create strong connections between the invented URIs and existing linked open databases. Below you can see the variety of the terms we used for each object defined in RDF. We also used GN and AAT codes for describing subjects and places.

  • Person: Roald Dahl

    • foaf:isPrimaryTopicOf https://www.imdb.com/title/tt5907158/": Meaning that we linked the URI of Roald Dahl to IMBd, an online film and audiovisual material database.

    • owl:sameAs http://viaf.org/108159131: Meaning that we connected the URI of Roald Dahl to the Virtual International Authority File.

    • core:notablyAssociatedWith person/quentin_blake: Meaning that we linked the URI of Roald Dahl to the URI of Quentin Blake.

  • Person: Quentin Blake

    • owl:sameAs http://viaf.org/108680806": Meaning that we linked the URI of Quentin Blake to the Virtual International Authority File

    • core:notablyAssociatedWith /person/roald_dahl: Meaning that we linked the URI of Quentin Blake to the URI of Roald Dahl.

  • Book: Matilda

    • owl:sameAs http://viaf.org/viaf/200813669: Meaning that the URI is connected to the Virtual International Authority File about Matilda.

    • Characters

      • Matilda: foaf:isPrimaryTopicOf https://en.wikipedia.org/wiki/Matilda_Wormwood: Meaning that the URI is connected to the Wikipedia page of the main character in the book.

      • Miss Trunchbull: foaf:isPrimaryTopicOf https://en.wikipedia.org/wiki/Miss_Trunchbull: Meaning that the URI is connected to the Wikipedia page of one of the characterâ€™s in the book.

  • Artefact: Sculpture

    • owl:sameAs https://statues.vanderkrogt.net/object.php?webpage=ST&record=gbse215": Meaning that we linked the URI of the artefact to the website depicting the same object.

    • foaf:primaryTopic /object/matilda: Meaning that we linked the URI of the artefact to the book which is its primary topic.
